# wbIsPseudoMediumMode

## Syntax

```pascal
function wbIsPseudoMediumMode: Boolean;
```

## Description

Returns whether pseudo Medium mode is enabled.

## Returns

Returns `True` if pseudo Medium mode is enabled, `False` otherwise.

## Example

```pascal
if wbIsPseudoMediumMode then
  AddMessage('Pseudo Medium mode is enabled');
```
