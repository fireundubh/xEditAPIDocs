# ChildGroup

## Syntax

```pascal
function ChildGroup(AWorldspace: IwbMainRecord): IwbGroupRecord;
```

## Description

Returns the group contained by `AWorldspace`

## Example

```pascal
grup := ChildGroup(wrld);
for i := 0 to Pred(ElementCount(grup)) do
	cell := ElementByIndex(grup, i);
```
