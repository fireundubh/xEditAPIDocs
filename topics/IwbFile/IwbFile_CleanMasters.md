# CleanMasters

## Syntax

```pascal
procedure CleanMasters(AFile: IwbFile);
```

## Description

Removes unused master files from `AFile`'s master files list

## Example

```pascal
f := FileByName('MyMod.esp');
CleanMasters(f);
```
