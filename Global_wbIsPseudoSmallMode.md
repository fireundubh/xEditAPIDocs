# wbIsPseudoSmallMode

## Syntax

```pascal
function wbIsPseudoSmallMode: Boolean;
```

## Description

Returns whether pseudo Small mode is enabled.

## Returns

Returns `True` if pseudo Small mode is enabled, `False` otherwise.

## Example

```pascal
if wbIsPseudoSmallMode then
  AddMessage('Pseudo Small mode is enabled');
```
