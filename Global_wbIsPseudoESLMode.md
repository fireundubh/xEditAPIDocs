# wbIsPseudoESLMode

## Syntax

```pascal
function wbIsPseudoESLMode: Boolean;
```

## Description

Returns whether pseudo ESL (Light) mode is enabled.

## Example

```pascal
if wbIsPseudoESLMode then
  AddMessage('Pseudo ESL mode is enabled');
```
